# Webshell-Sniper

```
一款基于终端的 Webshell 管理工具
```

#### 使用方法

```
python sniper.py [URL] [METHOD] [PASSWORD]
python sniper.py http://127.0.0.1/shell.php POST c
```

#### 功能介绍

1. 命令执行
2. 反弹 shell
3. 数据库管理
4. 一键搜索配置文件
5. 文件管理
6. 一键搜索 suid 程序
7. 一键植入 webshell
8. 一键植入内存木马
9. flag自动收割
10. 批量文件下载
11. 端口扫描
